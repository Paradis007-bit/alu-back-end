# API Project - Employee TODO List

## Description
This project contains a Python script that gathers data from a REST API for a given employee ID and displays their TODO list progress.

The script uses the **JSONPlaceholder** API and outputs the number of completed tasks, total tasks, and the titles of completed tasks in a formatted way.

---

## Files
- `api/0-gather_data_from_an_API.py` – main Python script to fetch and display TODO list information.
- `README.md` – this file.

---

## Usage
Run the script from the terminal with Python 3.4+:

```bash
$ python3 api/0-gather_data_from_an_API.py <employee_id>
